# Bank Loan Data Analysis

## Problem Statement
Analyze loan approvals and borrower risk patterns.

## Tools Used
SQL
Power BI

## Dataset
Loan application and borrower financial data.

## Analysis
- Loan approval trends
- Borrower risk segmentation
- Repayment patterns

## Key Insights
- Grade A and B borrowers have lower default risk
- Verified income borrowers show higher repayment reliability
